Sass

Sass es un pre-procesador de CSS que añade "super poderes" a tus estilos

Conceptos Básicos

  • Variables: Guardan valores reutilizables (colores, tamaños, etc.).
  • Anidamiento (Nesting): Escribe reglas CSS dentro de otras, reflejando la estructura HTML.
  • Mixins: Funciones que reutilizan bloques de estilos.
  • Herencia (@extend): Comparte estilos entre selectores.
  • Módulos (@use): Divide tu código en archivos parciales (importados con @use).
  • Parciales: Archivos Sass que comienzan con _ (ej: _variables.scss) y no se compilan a CSS.

CSS

  • Custom properties (variables): Son variables que almacenan valores reutilizables en CSS (como colores, tamaños o márgenes). Se definen con --nombre-variable y se usan con var()
  • CSS Nesting: Técnica para escribir reglas CSS dentro de otras, reflejando la estructura HTML
  • Funciones: Operaciones predefinidas que procesan valores directamente en CSS. Las más comunes
    • Matemáticas
      • calc
      • clamp
    • Colores
      • hsl
      • rgba
    • Transformación
      • rotate
      • scale

Componentes
