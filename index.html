Diseño Responsivo

Es un enfoque de diseño web que busca crear sitios web que se adapten a diferentes tamaños de pantalla y dispositivos, como computadoras de escritorio, tabletas y teléfonos móviles.

Mobile First

Es una estrategia de diseño que prioriza la creación de experiencias óptimas para dispositivos móviles antes de adaptarse a pantallas más grandes. En lugar de diseñar primero para computadoras de escritorio y luego ajustar el diseño para dispositivos móviles, se comienza con un diseño simple y funcional para móviles, y luego se expande a pantallas más grandes.

Breakpoint

Una medida que se utiliza para decirle al navegador que tiene que cambiar el layout (disposición) de los elementos. Se utilizan para indicar cómo queremos que se vea nuestra web en un dispositivo mobile, una tablet o un monitor de pc

Breakpoints más comunes

  • hasta 768px - celulares y tablets pequeñas
  • hasta 900px - tablets
  • desde 1200px - monitores

Custom Properties (variables de CSS)

Son una herramienta que permite crear cajas que guardan un valor que se va a reutilizar a largo de todo el proyecto. Son útiles para mantener la consistencia en el diseño y facilitar los cambios globales.

Utility class

Una clase de CSS pensada no para un elemento en particular sino para aplicar una serie de propiedades a varios elementos
