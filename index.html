Box Modeling

En CSS, el Box Modeling es una forma de pensar la estructura para realizar los diseños basada en cajas

las propiedades de box modeling no se heredan

  • Propiedades de la caja (cómo se forma la caja)
    • width
    • height
    • border
    • margin
    • padding
  • Forma en que se calcula el espacio ocupado por la caja
    • content-box (por defecto): El espacio final ocupado por la caja es el resultado de la suma de su contenido, width, padding, margin y border
    • border-box: El espacio final ocupado por la caja es igual al width
  • Características de la caja (tipo de display)
    • Elemento de línea (display: inline)
    • Elemento de bloque (display: block)
    • Elemento de línea-bloque (display: inline-block)
        • Display de "control de posición" - Se utilizan para modificar el layout de los elementos contenidos
      • Elementos flexibles (display: flex)
      • Elementos de grilla (display: grid)

Display Flex (flexbox)

Flexbox es un modelo de diseño unidimensional que permite alinear y distribuir espacio entre elementos en una interfaz de usuario.

Unidades de Medida

Son la herramienta que el navegador utiliza para calcular el tamaño que los distintos elementos deben tener

  • Absolutas - El tamaño es igual a la Medida
    • px
  • Relativas - El tamaño es el resultado del cálculo correspondiente a la medida utilizada
    • % - tamaño = ancho/alto del contenedor padre * %. Tamaño se calcula en relación al contenedor padre
    • vw/vh - tamaño = ancho(vw)/alto(vh) de la ventana del navegador. Tamaño se calcula en relación al navegador
      • svw - ancho del dispositivo más chico
      • dvw - ancho del dispositivo
    • em - tamaño = alto de la tipografía del contenedor padre * em. Tamaño depende de la tipografía del contenedor padre
    • rem - tamaño = alto de la tipografía del root (navegador). Tamaño depende de la tipografía del navegador

Se recomienda % y viewport para contenedores y rem para tipografías.
